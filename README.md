# DAISY

Programming language with Korean and English prose syntax.

## Quick Start
```
python tools/cli/daisy.py build examples/english_hello.dsy
python tools/cli/daisy.py run examples/english_hello.dsy
```

See `/docs` for detailed specs and interop guidance.

## Install
Windows:
```
.\scripts\install.ps1
```

macOS/Linux:
```
./scripts/install.sh
```

After install:
```
daisy --help
daisy build examples/english_hello.dsy
```

## Build Compiler
```
daisy build-compiler
```
